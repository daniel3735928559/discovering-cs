Every line in Python must be one of these:
  • Assignment:
    variable name = expression
  • Input number:
    input_num(variable name)
  • Input string:
    input_string(variable name)
  • Print:
    print(expression)
  • If:
    if(expression comparison expression):
  • Else:
    else:
  • While:
    while(expression comparison expression):
Where
  • A variable name can be any combination of letters,
    digits, and underscores except:
    • It may not begin with a number
    • It may not be a keyword
      (if, else, while, print, input)
  • An expression is any of the following:
    • A number (including decimals and negative numbers)
    • A string (anything contained in double-quotes)
    • A variable name
    • Any valid combination of the above three things using
      arithmetic operations:
      • + (addition)
      • - (subtraction)
      • * (multiplication)
      • / (division)
      • % (modulo)
