Language element Syntax Semantics Examples
Line Either an assignment line, a print line, or a flow-control line. A line is the basic unit of Python code and specifies a single command to be run. Each line in a Python program is run in sequence. 
x = 2
print('hello')
while(x == 2):
Assignment line 
variable name = expression
 Stores the value of the expression into a variable named by the given variable name 
x = 2
AbC_de5 = 'hi you!'
blah = 6*(2-1728)
Print line 
print(expression)
 Prints the value of the expression to the console 
print('hi world!')
print(2-2*7)
Flow-control line Either an if, an if/else, or a while Decides what line to execute next according to certain conditions 
if(x == 'no'):
while(y/2 >= x+1):
Variable name Any combination of letters, numbers, and underscores, except that a variable name cannot begin with a number or be a name that Python already treats as special (e.g. if, while, else--for a complete list, see the next section) Specifies the name of a variable. 
helo
abc1DEF0__0GHIJ

Non-examples: 
2hello
7
true
my name
Expression A number, string, variable name, or any valid combination of these using +, -, *, /, or %. Indicates a computation to do. The value of the expression is the result of the indicated computation. This value can either be a number of a string (but be aware that of course most operations do not make sense on strings--for example, 'hello'+3 is not allowed). 
'hello'
2*x+7
x/y
String Any sequence of characters (except a single-quote) enclosed in single-quotes. Strings provide a method to store and manipulate text. 
"Hello world!"
"\"Hmm\", he said"
if 
if(test): 
   indented lines
 Executes the indented lines if the test evaluates to true, and skips them if not
if/else 
if(test): 
   indented lines
else:
   indented lines
 Runs the first set of indented lines (after the 'if') and skips the second set (following the 'else') if the test evaluates to true. Otherwise, skips the first set and runs the second.
while 
while(test): 
   indented lines
 If the test evaluates to true, runs the indented lines. If when it gets to the end of the indented lines, the test still evaluates to true, starts over at the beginning of the indented lines.
Indented line A line can be indented from the previous line by putting three more spaces at the beginning of the line than were at the beginning of the previous line. A set of lines that are indented is a set of lines that go together--e.g. the indented lines following an if will all be skipped or will all be run, depending on the result of the test.
Test 
expr comparison expr
 A test is a way of comparing two expressions. If the comparison is correct, we say the test evaluates to true, otherwise, we say it evaluates to false. 
2*4 > x+1
y*y != x*x*x+2*x
Comparison Any of 
==, !=, >, <, >=, <=
 
==: Is equal to
!=: Is not equal to
>: Greater than
<: Less than
>=: Greater than or equal
<=: Less than or equal
